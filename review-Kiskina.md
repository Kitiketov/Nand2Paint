### названия переменных
1. main.jack
    ```c#
    function int binPow(int a, int n){
        var int p;
        if (n = 0){
            return 1;
        }
        if (n / 2 * 2 * -1 + n = 1){
            let p = Main.binPow(a, n - 1)*a;
            return p;
        }
        else{
            let p = Main.binPow(a, n / 2);
            let p = p * p;
            return p;
        }
    }
    ```
    с первого взгляда не очень понятно чего за ```a```, ```b``` и ```p```, можно чуть-чуть по-информативнее
